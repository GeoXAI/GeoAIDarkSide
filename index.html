About the Project

Uncertainty-aware Visual Analytics for GeoAI

GeoAI — the integration of machine learning and deep learning with geographic information — is reshaping how we understand and act on spatial phenomena. Yet beneath every prediction lies a hidden layer of uncertainty: uncertainty in model outputs, in spatial generalization, in feature attributions, and in the reliability of explanations across space. This is the dark side of GeoAI that practitioners rarely see.

This portal addresses the full spectrum of GeoAI uncertainty. That spans prediction uncertainty — how confident is the model at a given location? — through spatial calibration — do coverage guarantees hold under geographic heterogeneity? — to explainability uncertainty — how reliable are the feature attributions that explain a spatial prediction?

Each tool is built on the principle that uncertainty is not a defect to be hidden but a signal to be visualized. By making uncertainty a first-class citizen in the visual analytics pipeline, we enable analysts to reason about where a model is confident, why it errs spatially, and how to communicate those limitations to decision-makers.

Developer Guide

Adding a New Visualization

Extend the portal by appending an entry to the VISUALIZATIONS array in the <script> block of this file. No build step required. 

// ── Add a new visualization card ──────────────────────────
// Append to the VISUALIZATIONS array:

const VISUALIZATIONS = [
  /* ... existing entries ... */

  {
    id:     "my-new-viz",       // unique kebab-case slug
    title:  "My New Tool",       // card title
    desc:   "Short description of what this visualization does.",
    url:    "https://your-deployment-url.com/",
    tags: [
      { label: "Uncertainty", color: "red"  },
      { label: "Spatial AI",   color: "blue" },
    ],
    meta:   "Dataset or method note",  // small footer text
    icon:   "🗺️",                     // emoji shown in preview placeholder
    // Optional: provide a static screenshot for the preview thumbnail
    // preview: "assets/screenshots/my-new-viz.png",
  },
];
